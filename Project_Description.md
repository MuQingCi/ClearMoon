项目名称：ClearMoon

项目介绍：
基于Reactor多线程模型的高性能C++网络库，自下而上实现了基础组件层、Reactor网络层、HTTP服务层与异步日志系统。内核采用Epoll(LT)事件驱动 + 多IO线程池架构，主线程负责Accept分发，多个IO线程轮询处理读写事件，eventfd实现跨线程唤醒。支持HTTP协议解析与sendfile零拷贝文件传输。

项目亮点：
- 在AMD R7 7745HX + CPU绑定的压测环境下，/hello接口极限QPS达到162,281（800并发，延迟5.22ms），500~800并发为最佳工作区间，QPS稳定15.4万+
- 内置sendfile零拷贝机制，静态文件直接从内核态发送，无用户态内存拷贝，适用于高并发静态资源服务
- 核心实现约2,000行代码，架构清晰，无第三方依赖

技术栈：
C++11/14、Epoll(LT)、Reactor多线程、sendfile零拷贝、RAII
